# 🧿Mini Affirmation Generator
*This is a beginner-friendly Python program that generates positive affirmations based on user input*
---

##  ***✨Features***
**When the program is run, it displays a positve affirmation based on a user selected mindset category.**

- *The program organizes affirmations into catergories using a Python dictionary. Each category contains a list of affirmations related to a specific mindset category. When a category is selected, the program uses Python's built-random functionality to choose one affirmation from the corresponding list and display it.*
---
## ***🧠Skills Used***
- Python fundamentals
- Dictionaries and lists
- Functions
- Random selection using the 'random' module
- Command-line program structure
- Clean and readable code organization
---
## ***▶️How to Run***
*From project folder : python3 mini_affirmation_generator*
 1. User will be prompted to make a selection from the list of affirmation categories displayed
 2. To select the category, the user will need to type the corresponding number
 3. The program will generate and display one affirmation from the selected category
