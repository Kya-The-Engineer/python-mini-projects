# Mini Affirmation Generator
A beginner-friendly Python program that displays positive affirmations based on a category of your choosing or at random.

## What This Project Does
** When the program is run, it displays a positve affirmation based on a user selected mindset category **

## How It Works
1. Affirmations are stored in a dictionary 
2. A category is selected
3. One affirmation is randomly chosen and displayed

python mini_affirmation_generator.py
